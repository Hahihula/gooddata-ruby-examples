=== Listing project dashboards

by Tomas Korcak

==== Problem
You would like to list dashboards programmatically.

==== Prerequisites
You have to have existing project with dashboard(s).

==== Solution

[source, ruby]
----
require 'gooddata'

GoodData.with_connection do |c|
  # List all dashboards
  dashboards = project.dashboards
  pp dashboards
end
----
