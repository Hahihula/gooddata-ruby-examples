=== Delete data in project
by Tomas Svarovsky

==== Problem
You need to delete all the data in all datasets in a particular project.

==== Solution

In recipe 

[source,ruby]
----
# encoding: utf-8

require 'gooddata'

GoodData.with_connection do |c|
  GoodData.with_project('project_pid') do |project|
    project.delete_all_data    
  end
end
----
